# 单词

## 第一天

1. head 网页头部

2. body 网页主体

3. title 标题或者作为属性鼠标悬浮时显示的文字

4. a 超链接的路径

5. target 目标

6. _blank 空网页

7. javascript:; 空脚本

8. src 图片的路径

9. href 超链接的路径

10. alt 图片加载错误时显示的文字

11. ul 无序列表

12. ol 有序列表

13. li 列表项

14. form 表单

15. action 提交给服务端的地址

16. method 提交方式

17. get  明文提交

18. post 密文提交

    `get和post的区别，（面试必问）`

    

19. id 唯一，==前端==使用的标识

20. name 不唯一，前后端都可以用的标识（表单）

    

21. checked 默认选中的单选/复选

22. selected 默认选择的下拉选项

23. radio 单选框

24. checkbox 复选框

25. select 下拉框

26. option 下拉选项

27. submit 提交按钮

28. reset 重置按钮

29. value 用户提交或选择的值，提交给后端获得的值

30. textarea 多文本框

31. rows 多文本框的高度，行数

32. cols 多文本框的宽度，列数

33. readonly 只读，不可修改，可以被提交

34. disabled 禁止，不可修改且不被提交

35. file 文件

36. multiple 在file或者select中允许多选

37. display 显示方式

38. inline 按行标签排列

39. block 按块标签排列

40. inline-block 按行块标签排列

41. none 不要，不显示，取消

42. text-decoration 下划线

43. color 字体颜色

44. background-color 背景颜色

45. padding 内间距

46. outline 外线，不占用空间

47. box-sizing 占用空间的方式

48. border-box 不额外拓展空间

49. font-size 字体大小

50. position 定位

51. fixed 固定定位

52. border-radius 圆角

53. cursor 鼠标指针

54. pointer 手型指针

55. background-image 背景图片

56. background-size 背景图片大小

57. absolute 绝对定位

58. relative 相对定位

58. hover 鼠标悬停

59. focus 焦点

59. placeholder 文本框的默认提示

59. ::after 加在最末尾的伪元素

59. ::before 加在最前置的伪元素

64. transition 过渡

65. rotate 旋转

66. scale 放大缩小

67. translate 移动

68. animation 动画

69. @keyframes 定义动画

70. infinite 动画无限循环

71. forwards 动画保留到最后一帧

72. z-index 定位覆盖的层级，值大覆盖值小

73. font-weight 字体重量

74. bolder 加粗

75. flex 弹性布局

76. justify-content 弹性布局子节点的排列方式

77. flex-between 子节点间距自动排列

78. flex-around 子节点间距自动排列，两端头尾留出空隙

79. background-position背景图片位置

80. contain 背景图片等比列缩放，直到完全显示

    cover 背景图片等比例缩放，直到完全覆盖盒子，可能会有裁剪

81.  ! important 强制提升至最高优先级
