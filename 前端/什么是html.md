# 什么是HTML

超文本标记语言，用于网页开发的一种语言，也可以用于手机和PC的嵌入网页。



8位 = 1字节

1024字节 = 1Kb

1024kb = 1mb

1024mb = 1Gb

1024Gb = 1Tb

## 标签

有小于号和大于号包裹的

* ==行标签==，定义	：可以共享一行（不换行），但不能设置宽高

  * ==a==超链接
    * text-decoration:none; 可以删除下划线
  
  
    - ==span==容器
  
* ==块标签==，定义：可以设置宽高，但会换行

  - ==div== 容器

  - ==ul== 容器
  - ==li== 容器

* ==行块标签==，定义：可以共享一行，也可以设置宽高
  * `input`输入框，单选框，复选框
  * `img` 图片
    * `alt`属性：图片加载不出时显示
    * `title`属性：鼠标悬停时显示 




## 语法

文件后缀必须是：==.html==

==双标签==语法：<标签名称 属性1名称 = "属性1值" 属性2名称  = "属性2值" ..	> </标签名称>

例:  

```html
<a href= "http://www.baidu.com">点击跳转百度</a>
```



==单标签==语法：<标签名称 属性1名称 = "属性1值" 属性2名称  = "属性2值" ..	> 

例：`<br>`

# 什么是CSS

层叠式样式表，美化HTML组件，让HTML更好看。

+ css引入方式
+ 选择器：选中到你要美化的DOM节点
+ 定位：让DOM在特殊位置上摆放
+ 弹性布局：每个菜单项之间的间隔的控制（`间距相同`）
+ 盒子模型：调整盒子与盒子之间的距离，调整盒子与内部内容的距离，边框
+ 背景图片：设置背景图的位置
+ 字体：大小，颜色
+ 动画：伪类，伪元素，关键帧
  + ==tranform== 变换（一定要有==transition==)：下面属性用空格隔开
    + ==rotate/rotate3d== 旋转
    + ==scale== 放大缩小
    + ==translate== 移动




## CSS引入样式

1. 行内样式：不能复用，优先级最高，一般用的少，不推荐

   例：

   ```html
   <div style="color:red size:50px	"></div>
   ```

2. 内部样式：写在html文件中，但是是独立的一块空间，不和DOM耦合

   例：

   ```html
   <head>
       <style>
           #lay{
               color:red;
           }
       </style>
   </head>
   <body>
       <div id="lay">内容</div>
   </body>
   ```

   

3. 外部样式：把内部样式独立到单独的css文件中，推荐

   创建单独的css文件，比如==index.css==

   ```css
   #lay{color:red	}
   ```

   然后再html中引入该css文件

   ```html
   <head>
       <link href="index.css" rel="stylesheet"> 
   </head>
   <body>
       <div id="lay">
           内容
       </div>
   </body>
   ```

## 盒子模型

+ ==margin== 盒子与盒子之间的间隔

+ ==padding== 内容和盒子之间的距离，会改变盒子大小

  ```html
  #box{
  	box-sizing:border-box;		/*该属性能使padding和border从盒子内部计算，不向外拓展*/
  }
  ```

+ 块和行块使用==padding== 正常显示

+ 行标签使用==padding==和==margin==左右是正常显示的，上下在文档流中会不正常显示

+ 块标签上下==margin== 不累加，左右会累加

+ 总结
  + 嵌套情况用==padding==，独立情况用==margin==
  + 上下左右块级没问题，行级使用上下有问题
  + 方向距离总在一边调





##  定位==position==

+ 固定定位==fixed==:	固定在浏览器上的位置，不受滚动条影响

+ 相对定位==relative==：相对自己原始位置微调，需要注意==文档流不变==，只是视觉效果改变

+ 绝对定位==absolute==：固定在html文档上的位置，受滚动条影响

  + 绝对定位在定位的时候会依次向上查找最近==有定位的==祖先节点，
