## word

### 删除ai生成内容的中文与英文之间的空格

1. 按 `Ctrl+H` 打开替换窗口。
2. 点击 **更多** → 勾选 **使用通配符**。
3. 分别执行两次替换：

| 场景                   | 查找内容             | 替换为 |
| :--------------------- | :------------------- | :----- |
| 中文 + 空格 + 英文     | ([一-龥]) ([a-zA-Z]) | \1\2   |
| 英文 + 空格 + 中文     | ([a-zA-Z]) ([一-龥]) | \1\2   |
| 英文引号替换成中文引号 | (")([^"]*)(")        | “\2”   |
|                        |                      |        |
