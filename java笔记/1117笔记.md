@IntrinsicCandidate
public static int compareTo(byte[] value, byte[] other) {
    int len1 = value.length;//CDF
    int len2 = other.length;//CFG
    return compareTo(value, other, len1, len2);
}

public static int compareTo(byte[] value, byte[] other, int len1, int len2) {
    int lim = Math.min(len1, len2);//3
    for (int k = 0; k < lim; k++) {
        if (value[k] != other[k]) {
            return getChar(value, k) - getChar(other, k);//67  69
        }
    }
    //长度不同时
    return len1 - len2;
}
```

```
public static char getChar(byte[] val, int index) {
    return (char)(val[index] & 0xff);
}
```

### 8 输入流 输出流

计算机存储数据，主要硬盘（放到） 内存（暂时存，计算机重启没有保存）

输入流 输出流是针对内存而言，因为cpu是直接操作内存的

读文件：输入流

写文件：输出流

字符流：传输时，以字符为单位,一个字符 默认是二个字节 ，速度快

字节流：传输时，以字节为单位，速度慢, 一个字节占八位

场景：如果是文本传输，优先字符，传输快，但是，像视频 图片等，内部是由字节组成的，

此时只能用字节传输
