### 4.HashTable

- ```
  public class Hashtable<K,V>
  extends Dictionary<K,V>
  implements Map<K,V>, Cloneable, Serializable
  ```

hashTable不能设置null key和null 值

- `Hashtable`是同步的，是线程安全的
- hashTable初始容量是11，负载0.75

```

    this(11, 0.75f);
}
```
