### 2.同一个对象序列化多次

否

```
多个不同对象，每个对象 序列化存入，和取都是正常的，
相同的对象，多次存入，只序列化了一次
```

### 9.Thread Runnable

二者区别和联系

1.Thread是类，Runnable是一个接口

2.Thread有start方法  Runnable没有start

3.Thread还有很多其它的方法，Runnable只有一个run方法

为什么有了Runnable还须要Thread

因为Runnable没有启动线程的方法，所以要Thread

为什么有了Thread,还须要Runnable

因为要使用Thread方式，要通过extends Thread.

java是单继承，有时如果继承了其它类，继承不了Thread.

所以此时要通过实现Runnable

class Stu extends Person implements Runnable

场景

1 如果已extends了其它类，要使用线程，要使用Runnable

2 如果没有extends，优先使用Thread
