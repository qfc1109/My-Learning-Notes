### 5.==用数组完成质数的问题！！！==day03

思路：

5.1 每个数做个判断，如果是质数

5.2 把每个数，经过判断，存到一个数组中，再输出存好的数组

```java
 //用数组，输出0.200之间的质数
        int[] arr = new int[0];
        for (int i = 2; i < 201; i++) {//从2开始遍历
            boolean is = true;
            for (int j = 2; j <= i / 2;j++ ) {//
                if(i%j ==0){//只要有能整除的，就不是质数
                    is = false;
                    break;//只要有一个能整除就直接退出循环，提高效率
                }
            }
            if(is){
                arr = Arrays.copyOf(arr, arr.length+1);
                arr[arr.length-1] = i;
            }
        }
        System.out.println(Arrays.toString(arr));
```
