### 8.构造器

构造方法：

1.要==和类名相同==

2.写到类中，

3.是一个一般方法，方法名前没有返回类型,不用写void 也不用int

public static void xx(){

}

也不能写return

4.构造器可以重载，到时实例化时，会调用构造器，按参数去调用

5.每个类，会有默认构造器

默认构造器能删除吗？

1.当只有用到默认构造器，可以删除的

2.当有有参构造器，用并到默认构造器，此时不能删除

3.当有有参构造器时，没有用到默认构造器，此时可以删除
