逻辑关系：

&& 并 (与)

语句1  && 语句2  &&  语句3

如果每个都是真，最后才是真

&& 并  短路

语句1  && 语句2  &&  语句3 如果第一个已是false,后面不会走，

&  并  非短路 

语句1  && 语句2  &&  语句3  如果第一个已是false,后面还会走



或 ||   语句1  || 语句2  

只要其中的一句是真，就是真

||   短路 ，只要一个是真，后面就不会走，

|  不短路，每个都会执行
