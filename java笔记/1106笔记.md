深克隆：会创建一个新对象出来，信息也是相同的，如果此对象内部还有引用变量时，

课堂练习：

老师信息：id  name  String[] addrs

使用克隆，分别分用 浅 和深复制 来完成，并要证明浅和深的不同

```java
//重写clone方法--浅复制
@override
public Object clone() throws CloneNotSupportedException{
	return super.clone();
}

public Object clone() throws CloneNotSupportedException{
    Teacher t1 = new Teacher(id,name,addrs);//创建要复制的老师副本
    String[] arr = new String[addrs.length];//创建一个新的数组存放
    System.arraycopy(addrs,0,arr,0,addrs.length);//把旧数组放入新数组arr中
    t1.setAddrs(arr);
    return t1;
}
```
