### 4.自定义异常

目前Exception，是可以通过throw传入一个参数，参数内容可以赋我们写的内容

如果须二个参数，一个是内容，一个是时间

解决方案：

1 先定义一个类，extends Exception 重写，或自己定义方法也可以。

2 自定义异常：自己定义一个异常类，extends Exception

最后重写

`getMessage()` 

```
package com.day1105;

public class E5 {
    public static void main(String[] args) {
     try{
         int[] arr=new int[3];
         getArr(arr,-1);
     }catch (MyExe e){
         System.out.println(e.getMessage());
     }
    }
    public static void getArr(int[] arr,int index) throws MyExe {
        if(index<0){
            throw new MyExe("数组下标不能为负数","2025-11-05");
        }
    }
}

//自定义异常
class MyExe extends Exception {
    String cont;
    String time;
    public MyExe(){

    }
    public MyExe(String cont, String time) {
        this.cont = cont;
        this.time = time;
    }
    //重写getMessage()
    public String getMessage(){
        return this.time+" "+this.cont;
    }
}
```

开发思路：

1.写异常类时，重点管输出信息，比如重写

```
public String getMessage(){
    return this.time+" "+this.cont;
}
```

2.extends Exception可以自己选择

3.要业务本身的代码，要有方法内的   throw new MyExe(xx,yy)

4.哪个调用的地方，try catch()  ,在catch时，输出   e.getMessage()
